# API Example Responses

These files contain sample JSON responses for the public `v1` API endpoints.

## Files

- `agencies-list.json` - `GET /api/v1/agencies`
- `agency-detail.json` - `GET /api/v1/agencies/{slug}`
- `vendors-list.json` - `GET /api/v1/vendors`
- `vendor-detail.json` - `GET /api/v1/vendors/{slug}`

The payloads are based on the current Laravel resources and pagination format used by the backend.
